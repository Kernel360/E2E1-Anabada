## 기술 스택

- Backend
    - Java 17
    - Spring Boot 2.7
    - Spring Data JPA / QueryDSL
    - Spring Security / JWT / OAuth 2.0 Client
    - Spring Cloud OpenFeign
- Frontend
    - HTML / CSS
    - Javascript / Jquery
- DB
    - MySQL
- Cache
    - Spring Data Redis
